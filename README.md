# Course Learning Hub (課程學習中心)

Course Learning Hub 是一個基於 Node.js 本地伺服器的課程與教材管理系統，專為講師與學員設計。本系統結合了簡報線上播放、教材下載及完備的管理工具，並針對行動裝置進行了全面響應式優化（Mobile Friendly）。

---

## 🌟 系統特色

### 1. 學員端 (Student View)
*   **課前準備 (Prep Tab)**：公開開放，提供學員在課前瀏覽簡報 PDF、下載安裝教學或存取外部安裝連結（Reference Links）。
*   **正式課程 (Content Tab)**：
    *   **安全保護**：具備密碼保護機制，學員需輸入由講師提供之專屬課程密碼以解鎖完整課程。
    *   **無密碼公開課程**：講師可設定不啟用密碼，讓學員直接存取所有內容。
*   **簡報線上瀏覽 (PDF Slide Viewer)**：整合 PDF.js 投影片線上瀏覽、觸控滑動（Swipe）與播放幻燈片功能。
*   **檔案與參考連結清單**：清晰區分下載檔案與外部參考連結（附帶圖標與說明），並繞過下載安全條款直接開啟外連。

### 2. 管理工具 (Admin Tools)
*   **驗證登入**：採用密碼保護（密碼為 `admin123`）。
*   **課程建立與編輯完全自動化**：
    *   **建立課程**：自動在 `courses/` 下根據密碼 SHA-256 雜湊或 Course ID 建立實體目錄、自動保存上傳教材，並將中介資料寫入 `config.js` 檔案。
    *   **編輯既有課程**：允許直接更改講師、時數、日期與自訂「課程簡介」，並支援**無損儲存（在未解鎖正式課程下，更新基本資訊不會遺失已存正式教材）**。
    *   **密碼變更與目錄遷移**：修改課程密碼時，後端自動重新命名雜湊資料夾並遷移檔案，無縫移轉。
*   **教材與連結統一管理**：
    *   **直接上傳檔案**：檔案會被自動二進位讀取並在伺服器端寫入實體磁碟。
    *   **新增參考連結**：提供名稱、URL（自動格式化 URL 協定）及說明的填寫。
    *   **智慧同步與預設**：未解鎖正式課程時，新增教材自動預設為「課前準備 (Prep)」，便於立即編輯。
*   **一鍵排序 (Material Reordering)**：每行教材支援 ▲（上移）與 ▼（下移）以自訂學員端顯示順序。
*   **一鍵刪除課程**：徹底清理該課程在 config.js 中的中介資料與硬碟上的實體檔案。

## 📂 目錄結構

```text
├── courses/               # 存放所有課程資料（以 Course ID 或密碼 SHA-256 雜湊命名）
│   └── [course-id-or-hash]/
│       ├── prep/          # 課前準備公開教材
│       │   └── downloads/
│       ├── downloads/     # 正式課程私有教材
│       └── data.json      # 正式教材配置與簡報設定
├── index.html             # 單頁應用前端入口
├── app.js                 # 前端核心交互邏輯與 API 串接
├── styles.css             # 響應式與高級動態 UI 樣式表
├── server.js              # Node.js 本地後端服務 (MIME 支援、API 路由與檔案持久化)
├── config.js              # 全域公開課程中介資料 (COURSE_CATALOG)
└── README.md              # 說明文件
```

## 🚀 啟動與執行

### 本地開發環境啟動
本專案**無任何外部套件依賴**，只需擁有 Node.js 環境即可一鍵啟動：

```bash
# 啟動伺服器
node server.js
```
伺服器將在 `http://localhost:8080` 啟動，並會自動為您在預設瀏覽器中開啟首頁。
